# 📊 Airbnb Review Dashboard

Dashboard interaktif untuk menganalisis data ulasan (reviews) Airbnb menggunakan **Streamlit**.

---

## ✨ Fitur Utama
- Tren jumlah review per bulan/tahun
- Heatmap review berdasarkan hari dan bulan
- Distribusi review per tahun dan bulan
- Top 10 kata paling sering muncul di ulasan
- Word Cloud + Analisis Sentimen (Positif, Negatif, Netral)
- Filter data berdasarkan tahun
- Tabel data mentah + tombol download CSV

---

## 🛠️ Persyaratan

- **Python 3.8** atau lebih baru
- Sistem operasi: Windows, macOS, atau Linux
